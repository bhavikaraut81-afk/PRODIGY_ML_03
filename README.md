# 🐱🐶 Cats vs Dogs Classification using SVM

## 📌 Overview
This project classifies images of cats and dogs using Support Vector Machine (SVM).

## ⚙️ Technologies
- Python
- OpenCV
- Scikit-learn

## 🧠 Approach
- Image preprocessing (resize + grayscale)
- Feature extraction (flattening)
- PCA for dimensionality reduction
- SVM classification

## 📊 Results
- Accuracy: ~65–75%

## 🚀 How to Run
pip install -r requirements.txt  
python main.py
