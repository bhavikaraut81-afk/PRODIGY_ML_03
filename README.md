# 🐱🐶 Cats vs Dogs Classification using SVM

## 📌 Overview
This project classifies images of cats and dogs using a classical Machine Learning approach (Support Vector Machine).

Instead of deep learning, this project demonstrates how traditional ML techniques can be applied to image classification tasks.

---

## ⚙️ Technologies Used
- Python
- OpenCV
- NumPy
- Scikit-learn
- Matplotlib

---

## 🧠 Approach

1. Image Collection (downloaded using script)
2. Image Preprocessing:
   - Convert to grayscale
   - Resize to 64x64
3. Feature Extraction:
   - Flatten image pixels
4. Dimensionality Reduction:
   - PCA (Principal Component Analysis)
5. Model Training:
   - Support Vector Machine (SVM)
6. Prediction & Visualization

---

## 📊 Results

- Accuracy: ~60–70%
- Model predicts whether an image is a cat or a dog
